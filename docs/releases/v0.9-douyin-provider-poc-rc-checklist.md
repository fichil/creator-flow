# v0.9 Douyin Provider POC Release Candidate Checklist

This checklist prepares the v0.9 Douyin Provider POC / Sandbox Integration release candidate review package. It is not a final release declaration, does not create a tag, and does not mean v0.9 POC is complete.

## Scope Confirmation

- [ ] v0.9 is Douyin Provider POC / Sandbox Integration.
- [ ] Current v0.9 supports sandbox-only deterministic simulation.
- [ ] Backend adapter skeleton exists for `douyin_sandbox` and `douyin_real`.
- [ ] Sandbox operations return deterministic simulated / dry-run results.
- [ ] Provider registry / factory routing exists for exact whitelisted provider ids.
- [ ] Sandbox metrics / mock workflow exists.
- [ ] Sandbox-only backend API contract exists.
- [ ] Frontend sandbox POC panel exists.
- [ ] `douyin_real` remains blocked / not implemented.
- [ ] Unknown provider ids fail explicitly and do not fallback.
- [ ] v0.9 does not connect to real Douyin.
- [ ] v0.9 does not implement OAuth.
- [ ] v0.9 does not create OAuth URLs.
- [ ] v0.9 does not save tokens, secrets, or credentials.
- [ ] v0.9 does not upload, publish, schedule, or fetch real metrics.
- [ ] v0.9 does not claim production readiness.
- [ ] v0.9 does not claim commercial readiness.

## Test Matrix

- [ ] Full backend tests pass.
- [ ] Full frontend tests pass.
- [ ] Frontend build passes.
- [ ] `scripts/smoke-api.ps1` passes against a local backend.
- [ ] `git diff --check` passes.
- [ ] `git status --short` is reviewed before commit and empty after commit.
- [ ] Safety / secret scan is reviewed.
- [ ] Sensitive payload scan is reviewed.
- [ ] External call boundary is reviewed.
- [ ] Documentation wording scan is reviewed.
- [ ] Route / API boundary scan is reviewed.
- [ ] Frontend UI boundary scan is reviewed.

## API Readiness

- [ ] Provider list endpoint returns `douyin_sandbox` and `douyin_real`.
- [ ] Provider lookup endpoint returns exact whitelisted providers only.
- [ ] Mock connection endpoint returns deterministic sandbox simulation.
- [ ] Metrics preview endpoint returns deterministic sandbox metrics preview.
- [ ] Publish dry-run endpoint returns deterministic sandbox publish dry-run.
- [ ] `douyin_real` remains blocked through API responses.
- [ ] Unknown provider ids do not fallback.
- [ ] API results are deterministic.
- [ ] API performs no DB mutation.
- [ ] API performs no external call.
- [ ] API returns no sensitive payload.

## Frontend Readiness

- [ ] Sandbox panel renders on the project list page.
- [ ] Boundary banner is visible.
- [ ] Provider descriptors are displayed.
- [ ] Mock connection action is sandbox-only.
- [ ] Metrics preview action is sandbox-only.
- [ ] Publish dry-run action is sandbox-only.
- [ ] Error handling is sandbox-safe.
- [ ] No OAuth UI is present.
- [ ] No token, secret, or credential input is present.
- [ ] No upload, publish, or schedule UI is present.
- [ ] No production or commercial readiness wording is displayed as a current capability.

## Release Candidate Exit Criteria

- [ ] All tests pass.
- [ ] Smoke API passes or any environment limitation is documented.
- [ ] No whitespace errors remain.
- [ ] No uncommitted changes remain after commit.
- [ ] No real secrets are present.
- [ ] No ignored runtime artifacts are committed.
- [ ] No false production or commercial readiness claims are present.
- [ ] No real Douyin, OAuth, token, publish, or metrics implementation is added.
- [ ] Final branch is pushed.
- [ ] Next step is human review / PR / merge decision, not automatic release.

## Non-Goals

- Real Douyin API.
- Real OAuth.
- OAuth callback.
- OAuth state storage.
- Token exchange.
- Token storage.
- Credential storage.
- Database schema change.
- Real metrics.
- Real upload.
- Real publish.
- Real schedule.
- Tenant / billing / RBAC / admin console.
- Production deployment.
- Direct-customer commercial release.
- Customer-of-customer SaaS commercial release.
