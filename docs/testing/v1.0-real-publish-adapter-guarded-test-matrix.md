# v1.0 Real Publish Adapter Guarded Test Matrix

This matrix covers the v1.0 Batch 7 guarded real publish adapter foundation. It does not require real Douyin, OAuth URLs, OAuth start routes, OAuth callback routes, real token exchange, real token storage, real credential storage, upload, actual publish, scheduled publish, metrics, or external service calls.

Batch 8 publish status reconciliation may depend on Batch 7 metadata-only publish attempts, but that dependency does not convert a local attempt into a real provider publication. Batch 9 limited metrics snapshots may depend on Batch 8 metadata-only status snapshots, but that dependency does not convert local status metadata into real Douyin metrics. Batch 8 and Batch 9 remain local-only and metadata-only; real Douyin status query and real metrics read still require later accepted implementation batches.

| Scenario | Expected result | Safety checks |
| --- | --- | --- |
| Confirmed publish intent required | Missing, cancelled, blocked, or non-confirmed intent is rejected. | No publish attempt row is created. |
| Review state linked | Attempt must reference the same project review item. | Review item must remain approved. |
| Preflight validation | Review, media, metadata, provider, credential reference, and duplicate checks run before local attempt creation. | Failure categories stay safe and redacted. |
| Media readiness | Missing local media artifact returns `media_not_ready`. | No upload is attempted. |
| Metadata readiness | Sensitive markers or malformed metadata return `metadata_invalid`. | No token, secret, credential, code, state, or response material is emitted. |
| Credential reference readiness | Missing safe credential reference returns `credential_reference_missing`. | No real credential storage is enabled. |
| Duplicate attempt prevention | Active attempt for the same publish intent is rejected. | Existing local attempt is not executed again. |
| Real provider disabled | `douyin_real` remains blocked by default. | Batch 5 controls remain blocking. |
| Kill switch active | Kill switch active returns `kill_switch_active`. | Kill switch takes precedence over publish attempt execution. |
| Platform preconditions missing | Missing platform preconditions return `platform_preconditions_missing`. | No real call path opens. |
| Sandbox fallback forbidden | Real provider cannot fallback to `douyin_sandbox`. | Result records no fallback. |
| Unsupported provider | Unknown provider returns `unsupported_provider`. | Unknown id is not mapped to sandbox or fake. |
| No external service | Workflow does not call socket, requests, httpx, urllib, Douyin, or business external services. | Network monkeypatches remain unused. |
| No upload | Attempt creation does not upload media. | Upload runtime paths remain untouched by guarded attempts. |
| No actual publish | Attempt creation does not call provider publish. | Existing fake execution is not triggered by guarded attempts. |
| No scheduled publish | Attempt creation does not create scheduled publish records. | No scheduled publish route is added. |
| No OAuth route | Batch 7 adds no OAuth start, callback, URL creation, or token exchange route. | Route list has no new OAuth runtime endpoints. |
| No token exchange | No exchange runtime is invoked. | No authorization code or token material is returned or persisted. |
| No credential storage | No real credential storage is enabled. | Publish attempt schema has no credential columns. |
| No metrics read | Attempt creation does not fetch metrics. | Metrics routes are not invoked by guarded attempt creation, and Batch 9 local metrics snapshots do not enable real metrics reads. |
| Batch 8 status reconciliation dependency | Publish status reconciliation must require an existing Batch 7 publish attempt. | Batch 8 still does not query real Douyin status, persist raw status responses, or read metrics. |
| Batch 9 metrics snapshot dependency | Limited metrics snapshots must require an existing Batch 8 status snapshot. | Batch 9 still does not query real Douyin metrics, persist raw metrics responses, or query real status. |
| UI wording | UI says guarded/local publish attempt, not published. | UI does not claim upload, scheduled publish, or real publish succeeded. |
| Redaction | Responses, messages, fixtures, and repr output omit sensitive material values. | No token, secret, credential, authorization code, raw OAuth state, cookie, session, API key, bearer, raw request, raw response, provider response, upload response, publish response, external response, or Douyin response values. |
