# v1.0 Limited Metrics Read Guarded Test Matrix

This matrix covers the v1.0 Batch 9 limited metrics read guarded foundation. It does not require real Douyin, OAuth URLs, OAuth start routes, OAuth callback routes, real token exchange, real token storage, real credential storage, upload, actual publish, scheduled publish, real status query, real metrics query, or external service calls.

| Scenario | Expected result | Safety checks |
| --- | --- | --- |
| Status snapshot required | Missing status snapshot returns `publish_status_required`. | No metrics snapshot row is created. |
| Metadata-only metrics snapshot | Valid local status snapshot creates a metrics snapshot. | Snapshot stores safe counts, freshness, source, timestamps, and safe messages only. |
| Metrics freshness visible | Snapshot response includes `metrics_freshness_status`. | UI displays fresh, stale, unknown, or not available without claiming real platform data. |
| Metrics permission missing | Missing permission metadata returns `metrics_permission_missing`. | No provider metrics query is attempted. |
| Malformed fixture rejected | Invalid fake or sandbox fixture returns `metrics_fixture_invalid`. | Fixture raw payload is not persisted or echoed. |
| Real provider disabled | `douyin_real` remains blocked by default. | Batch 5 controls remain blocking. |
| Kill switch active | Kill switch active returns `kill_switch_active`. | Kill switch takes precedence over metrics execution. |
| Platform preconditions missing | Missing platform preconditions return `platform_preconditions_missing`. | No real call path opens. |
| Sandbox fallback forbidden | `douyin_real` cannot fallback to `douyin_sandbox`. | Result records no fallback. |
| Unsupported provider | Unknown provider returns `unsupported_provider`. | Unknown id is not mapped to sandbox or fake. |
| No external service | Workflow does not call socket, requests, httpx, urllib, Douyin, or business external services. | Network monkeypatches remain unused. |
| No real metrics query | External metrics query request returns `external_metrics_query_blocked`. | External query status stays `not_called` or `blocked`. |
| No real status query | Metrics snapshot creation does not query real publication status. | Status snapshot dependency is local metadata only. |
| No upload | Metrics snapshot creation does not upload media. | Upload runtime paths remain untouched. |
| No actual publish | Metrics snapshot creation does not call provider publish. | Existing fake publish execution is not triggered. |
| No scheduled publish | Metrics snapshot creation does not create scheduled publish records. | No scheduled publish route is added. |
| No OAuth route | Batch 9 adds no OAuth start, callback, URL creation, or token exchange route. | Route list has no new OAuth runtime endpoints. |
| No OAuth URL | UI and API do not open or create authorization URLs. | No browser navigation or URL payload is produced. |
| No token exchange | No exchange runtime is invoked. | No authorization code or token material is returned or persisted. |
| No credential storage | No real credential storage is enabled. | Metrics schema has no credential columns. |
| UI wording | UI says local / limited / fake or sandbox-safe metrics snapshot. | UI does not claim real Douyin metrics were fetched or real performance was validated. |
| Redaction | Responses, messages, fixtures, and repr output omit sensitive material values. | No token, secret, credential, authorization code, raw OAuth state, cookie, session, API key, bearer, raw request, raw response, provider response, upload response, publish response, status response, metrics response, external response, or Douyin response values. |
