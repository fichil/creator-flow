# v1.0 User-Confirmed Publish Intent Test Matrix

This matrix covers the v1.0 Batch 6 local publish intent workflow foundation. It does not require real Douyin, OAuth URLs, OAuth start routes, OAuth callback routes, real token exchange, real token storage, real credential storage, real publish, upload, metrics, or external service calls.

| Scenario | Expected result | Safety checks |
| --- | --- | --- |
| Explicit confirmation required | Missing confirmation is rejected with `confirmation_required`. | No publish intent row is created. |
| Malformed confirmation | Non-boolean confirmation is rejected. | Response contains no sensitive material. |
| Review state linked | Publish intent requires an existing review item in the same project. | Cross-project review ids fail safely. |
| Review not ready | Pending or rejected review items are rejected. | Review approval is not equivalent to publish execution. |
| Preflight validation | Local media readiness and metadata validation run before creation. | Missing media returns `media_not_ready`; invalid metadata returns `metadata_invalid`. |
| Duplicate publish prevention | Active intent for the same review item/provider/source is rejected. | Cancelled intents are the only safe retry path. |
| Real provider disabled | `douyin_real` returns `real_provider_disabled`. | Batch 5 controls remain blocking. |
| Sandbox fallback forbidden | Real provider cannot fallback to `douyin_sandbox`. | Result records no fallback. |
| Unsupported provider | Unknown provider returns `unsupported_provider`. | Unknown id is not mapped to sandbox or fake. |
| Source separation | `fake_local` and `douyin_sandbox` stay distinct. | `douyin_real` remains blocked. |
| No external service | Workflow does not call socket, requests, httpx, urllib, Douyin, or business external services. | Network monkeypatches remain unused. |
| No upload | Creation does not upload media. | Upload runtime paths remain untouched by publish intent creation. |
| No real publish | Creation does not call provider publish. | Existing fake execution is not triggered by intent creation. |
| No OAuth route | Batch 6 adds no OAuth start, callback, URL creation, or token exchange route. | Route list has no new OAuth runtime endpoints. |
| No token exchange | No exchange runtime is invoked. | No authorization code or token material is returned or persisted. |
| No credential storage | No real credential storage is enabled. | Publish intent schema has no credential columns. |
| No metrics read | Creation does not fetch metrics. | Metrics routes are not invoked by publish intent creation. |
| UI wording | UI says Publish Intent / local intent, not published. | UI does not show real OAuth, upload, scheduled publish, or real publish wording. |
| Redaction | Responses, messages, fixtures, and repr output omit sensitive markers. | No token, secret, credential, authorization code, raw OAuth state, cookie, session, API key, bearer, raw request, raw response, provider response, upload response, or publish response values. |
