# v1.0 User Test Release Candidate Test Matrix

This matrix covers the v1.0 Batch 10 user test readiness / release candidate package. It does not require real Douyin, OAuth URLs, OAuth start routes, OAuth callback routes, real token exchange, real token storage, real credential storage, upload, actual publish, scheduled publish, real status query, real metrics query, or external service calls.

| Area | Expected evidence | Safety check |
| --- | --- | --- |
| Batch 1 acceptance | OAuth boundary / callback contract docs and matrix exist. | No route or real OAuth runtime is implied. |
| Batch 2 acceptance | OAuth state / anti-replay tests pass. | Raw state is not stored or exposed. |
| Batch 3 acceptance | Token exchange boundary tests pass. | No real token exchange or token material. |
| Batch 4 acceptance | Credential boundary tests pass. | No real token or credential storage. |
| Batch 5 acceptance | Real provider controls tests pass. | `douyin_real` blocked by default. |
| Batch 6 acceptance | Publish intent tests and UI pass. | Intent is local-only and user-confirmed. |
| Batch 7 acceptance | Guarded publish attempt tests and UI pass. | No upload, publish, or schedule. |
| Batch 8 acceptance | Status reconciliation tests and UI pass. | No real status query or raw status response. |
| Batch 9 acceptance | Limited metrics snapshot tests and UI pass. | No real metrics query or raw metrics response. |
| Backend route inventory | Route list reviewed. | No OAuth start / callback / URL route, no real publish, upload, schedule, status query, or metrics query route. |
| Frontend UI wording inventory | Project detail UI wording reviewed. | No claim that real Douyin OAuth, publish, status, or metrics occurred. |
| DB schema forbidden fields inventory | Schema scan reviewed. | No forbidden token, secret, credential, raw state, or raw response columns in local-only tables. |
| Redaction coverage | Tests and scans cover safe messages, repr, docs, and fixtures. | No sensitive value is returned, logged, persisted, or rendered. |
| No external service coverage | Network monkeypatch tests and validation scripts pass. | No business external service call. |
| No OAuth route coverage | Route scan passes. | No OAuth start, callback, URL creation, or token exchange route. |
| No OAuth URL coverage | UI and API scan passes. | No authorization URL is created or opened. |
| No real token exchange coverage | Boundary tests pass. | Only fake-gated metadata exists. |
| No real credential storage coverage | Boundary tests pass. | Only metadata-only references exist. |
| No real publish / upload / schedule coverage | Publish intent and guarded attempt tests pass. | No uncontrolled publish path. |
| No real status query coverage | Status reconciliation tests pass. | External status query stays blocked. |
| No real metrics query coverage | Metrics snapshot tests pass. | External metrics query stays blocked. |
| Real provider default blocked | Batch 5 / 7 / 8 / 9 tests pass. | `douyin_real` remains disabled. |
| Sandbox fallback forbidden | Batch 3-9 tests pass. | `douyin_real` cannot fallback to `douyin_sandbox`. |
| RC artifacts present | ADR 0055, RC checklist, release notes draft, PR draft, user guide, runbook, and RC matrix exist. | Artifacts state this is not production, commercial, or SaaS readiness. |
